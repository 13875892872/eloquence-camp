# 用户偏好

## 工作习惯
- 用户是项目所有者，关注功能完整性而非代码细节
- 偏好先设计方案再动手实现
- 需要中文回复和交流
- 每次改动后需要看到验证结果

## 技术偏好
- 优先复用现有代码和模式，不引入新依赖
- AI 服务统一用阿里云 DashScope（Qwen），不引入其他平台
- 前端风格保持简洁卡片式设计，用 Element Plus / uni-app 原生组件
- 数据库优先用 SQLite 开发，MySQL 生产

## 项目目标
- 快速迭代到可上线状态
- 素材丰富度是核心竞争力（每天自动获取）
- 小程序体验优先于管理后台
