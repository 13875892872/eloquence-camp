## 一、前置准备

### 1.1 必须准备的事项

| # | 事项 | 操作路径 | 周期 |
|---|------|---------|------|
| 1 | **微信小程序注册** | [mp.weixin.qq.com](https://mp.weixin.qq.com) → 注册 → 选择"小程序" | 1-3 天 |
| 2 | **备案域名** | 阿里云/腾讯云 → 域名注册 → 提交备案 | **15-20 天** |
| 3 | **SSL 证书** | 阿里云 SSL 证书（免费版即可）或 Let's Encrypt | 即时 |
| 4 | **云服务器** | 阿里云 ECS / 腾讯云 CVM（建议 2核4G，Linux） | 即时 |
| 5 | **MySQL 数据库** | 阿里云 RDS MySQL 8.0 / 自建 MySQL | 即时 |
| 6 | **OSS 对象存储** | 阿里云 OSS（存音频文件） | 即时 |

> ⚠️ **域名备案是最大瓶颈**，备案期间可以先完成所有技术部署和测试。

### 1.2 服务器建议配置

| 组件 | 规格 | 说明 |
|------|------|------|
| CPU | 2 核+ | Qwen API 不占用本机，Flask 轻量 |
| 内存 | 4 GB | Gunicorn 4 worker + MySQL |
| 系统盘 | 40 GB | 系统 + 代码 + 日志 |
| 数据盘 | 100 GB+ | OSS 上的音频文件 |
| 带宽 | 3 Mbps+ | 小程序主要为文本 API，音频走 OSS |
| 系统 | Ubuntu 22.04 / CentOS 8 | 推荐 Ubuntu |

### 1.3 域名规划

| 域名 | 用途 | SSL |
|------|------|-----|
| `api.your-domain.com` | 后端 API | ✅ 必需 |
| `admin.your-domain.com` | 管理后台（可选） | ✅ 推荐 |
| `cdn.your-domain.com` | OSS 绑定域名（可选） | ✅ 推荐 |

## 二、代码改动

### 2.1 小程序 API 地址（已改造为条件编译）

小程序中已有两处使用条件编译，开发构建自动走本地 IP，生产构建 `npm run build:mp-weixin` 自动走正式域名。

**`miniapp/src/api/request.js`**：
```js
// #ifdef MP-WEIXIN
const BASE_URL = 'https://api.your-domain.com/api'  // ← 替换为正式域名
// #endif
// #ifndef MP-WEIXIN
const BASE_URL = 'http://192.168.43.30:5000/api'
// #endif
```

**`miniapp/src/App.vue`**（登录接口）：
```js
function getBaseUrl() {
  // #ifdef MP-WEIXIN
  return 'https://api.your-domain.com/api'  // ← 替换为正式域名
  // #endif
  // #ifndef MP-WEIXIN
  return 'http://192.168.43.30:5000/api'
  // #endif
}
```

> 📌 发布前搜索所有 `your-domain.com` 替换为你的真实域名。

### 2.2 填写微信 AppID

**`miniapp/src/manifest.json`**：
```json
"mp-weixin": {
    "appid": "wx1234567890abcdef",   // ← 替换
    "setting": {
        "urlCheck": true,
        "es6": true,
        "postcss": true,
        "minified": true
    }
}
```

### 2.3 填写 `.env` 生产配置

编辑 `server/.env`（或单独创建 `.env.production`），填入真实值：

```bash
# ─── 环境 ───
FLASK_ENV=production
FLASK_DEBUG=False

# ─── 安全密钥（⚠️ 必须改为随机字符串）───
SECRET_KEY=生成一个随机长字符串
JWT_SECRET_KEY=生成另一个随机长字符串
JWT_ACCESS_TOKEN_EXPIRES=86400

# ─── 数据库 ───
DATABASE_URL=mysql+pymysql://用户名:密码@数据库地址:3306/eloquence_camp

# ─── 微信 ───
WECHAT_APPID=wx1234567890abcdef
WECHAT_SECRET=你的小程序Secret

# ─── AI 服务（阿里云 DashScope）───
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxxxxx
QWEN_TEXT_MODEL=qwen-plus
QWEN_FEEDBACK_MODEL=qwen-plus

# ─── OSS 对象存储 ───
OSS_ENDPOINT=oss-cn-hangzhou.aliyuncs.com
OSS_ACCESS_KEY_ID=你的AccessKeyId
OSS_ACCESS_KEY_SECRET=你的AccessKeySecret
OSS_BUCKET_NAME=eloquence-camp-prod

# ─── 语音评测（阿里云智能语音交互）───
ALIYUN_SPEECH_APP_KEY=你的语音AppKey
ALIYUN_SPEECH_ACCESS_KEY_ID=你的AccessKeyId
ALIYUN_SPEECH_ACCESS_KEY_SECRET=你的AccessKeySecret
```

> ⚠️ 随机字符串生成方式：`python -c "import secrets; print(secrets.token_hex(32))"`

## 三、服务器部署

### 3.1 基础环境

```bash
# SSH 登录
ssh root@your-server-ip

# 更新系统
apt update && apt upgrade -y

# 安装依赖
apt install -y python3 python3-pip python3-venv python3-dev \
    nginx mysql-client supervisor git

# 创建应用目录
useradd -m -s /bin/bash eloquence
mkdir -p /opt/eloquence-camp
chown eloquence:eloquence /opt/eloquence-camp
```

### 3.2 上传代码

```bash
# 从本地执行（在项目根目录）
rsync -avz --exclude 'venv' --exclude '__pycache__' --exclude '*.db' \
    server/ root@your-server:/opt/eloquence-camp/server/
```

### 3.3 安装 Python 依赖

```bash
# 在服务器上
cd /opt/eloquence-camp/server
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install gunicorn
```

### 3.4 初始化数据库

```bash
cd /opt/eloquence-camp/server
source venv/bin/activate

# 创建表 + 种子数据
PYTHONPATH=. python seeds/admin_seed.py
PYTHONPATH=. python seeds/config_seed.py
PYTHONPATH=. python seeds/demo_data_seed.py
```

### 3.5 配置 Gunicorn

创建 `/opt/eloquence-camp/server/gunicorn.conf.py`：

```python
import multiprocessing

# 绑定地址（Nginx 反向代理到 127.0.0.1）
bind = "127.0.0.1:5000"

# Worker 数量
workers = multiprocessing.cpu_count() * 2 + 1

# Worker 类型
worker_class = "sync"
worker_connections = 1000

# 超时
timeout = 60
graceful_timeout = 30

# 日志
accesslog = "/var/log/eloquence/access.log"
errorlog = "/var/log/eloquence/error.log"
loglevel = "info"

# 进程管理
pidfile = "/tmp/gunicorn_eloquence.pid"
daemon = False
```

创建日志目录：
```bash
mkdir -p /var/log/eloquence
chown eloquence:eloquence /var/log/eloquence
```

### 3.6 配置 Supervisor（进程守护）

创建 `/etc/supervisor/conf.d/eloquence.conf`：

```ini
[program:eloquence]
command=/opt/eloquence-camp/server/venv/bin/gunicorn -c /opt/eloquence-camp/server/gunicorn.conf.py wsgi:app
directory=/opt/eloquence-camp/server
user=eloquence
autostart=true
autorestart=true
redirect_stderr=true
stdout_logfile=/var/log/eloquence/supervisor.log
environment=PYTHONPATH=".",PYTHONIOENCODING="utf-8"
```

启动：
```bash
supervisorctl reread
supervisorctl update
supervisorctl start eloquence
```

### 3.7 配置 Nginx

创建 `/etc/nginx/sites-available/eloquence`：

```nginx
# ─── HTTP → HTTPS 重定向 ───
server {
    listen 80;
    server_name api.your-domain.com;

    location / {
        return 301 https://$host$request_uri;
    }
}

# ─── HTTPS 主配置 ───
server {
    listen 443 ssl http2;
    server_name api.your-domain.com;

    # SSL 证书
    ssl_certificate     /etc/ssl/certs/your-domain.pem;
    ssl_certificate_key /etc/ssl/private/your-domain.key;

    # SSL 安全配置
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256;
    ssl_prefer_server_ciphers on;
    ssl_session_cache shared:SSL:10m;
    ssl_session_timeout 10m;

    # 日志
    access_log /var/log/nginx/eloquence-access.log;
    error_log  /var/log/nginx/eloquence-error.log;

    # 上传文件大小限制（音频文件）
    client_max_body_size 20m;

    # API 反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        # 超时
        proxy_connect_timeout 60s;
        proxy_read_timeout 60s;
    }

    # 静态音频文件（OSS 回源或本地存储）
    location /api/upload/ {
        proxy_pass http://127.0.0.1:5000;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

启用配置：
```bash
ln -s /etc/nginx/sites-available/eloquence /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default  # 删除默认站点
nginx -t                                    # 测试配置
systemctl reload nginx                      # 重载
```

### 3.8 配置防火墙

```bash
# 仅开放必要端口
ufw allow 22    # SSH
ufw allow 80    # HTTP
ufw allow 443   # HTTPS
ufw enable
```

## 五、管理员后台部署

### 5.1 生产构建

```bash
cd admin

# 确认 .env.production 中 VITE_API_BASE_URL 指向正式 API
# VITE_API_BASE_URL=https://api.your-domain.com/api
```

### 5.2 部署到服务器

```bash
# 上传到 Nginx 静态目录
rsync -avz admin/dist/ root@your-server:/var/www/admin/

# 或部署到 CDN / OSS 静态托管
```

### 5.3 Nginx 配置（管理后台）

```nginx
server {
    listen 443 ssl http2;
    server_name admin.your-domain.com;

    ssl_certificate     /etc/ssl/certs/your-domain.pem;
    ssl_certificate_key /etc/ssl/private/your-domain.key;

    root /var/www/admin;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 代理（管理后台也通过此域名调用 API）
    location /api/ {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 六、定时任务

### 6.1 素材自动获取

在服务器上用 Linux cron 替代开发环境的 Windows 计划任务：

```bash
crontab -e -u eloquence
```

添加：
```
# 每天早上 8:00 自动获取训练素材
0 8 * * * cd /opt/eloquence-camp/server && PYTHONPATH=. /opt/eloquence-camp/server/venv/bin/python scheduler/run.py >> /var/log/eloquence/scheduler.log 2>&1
```

### 6.2 日志轮转

```bash
# /etc/logrotate.d/eloquence
/var/log/eloquence/*.log {
    daily
    rotate 30
    compress
    delaycompress
    missingok
    notifempty
    copytruncate
}
```

## 七、提交审核

### 7.1 提审前自检

在真实手机上用**体验版**完整测试：

- [ ] 微信登录正常（能获取到 code → 换取 token）
- [ ] 首页数据展示正常
- [ ] 打卡流程完整（查看任务 → 完成打卡 → 解锁下一个）
- [ ] 训练题库可浏览、可搜索
- [ ] 训练详情可查看范文
- [ ] TTS 范本播放正常
- [ ] 录音功能正常（真机测试）
- [ ] AI 文案生成正常（四种场景都试一遍）
- [ ] 个人中心数据正确
- [ ] 收藏功能正常
- [ ] 练习记录可查看

### 7.2 准备审核材料

| 材料 | 说明 |
|------|------|
| 测试账号 | 在数据库预留一个测试微信账号 |
| 功能截图 | 首页/打卡/训练/AI生成/个人中心各一张 |
| 隐私政策 | 注明收集微信头像昵称的用途 |
| 类目选择 | 建议选「教育 > 在线教育」或「工具 > 信息查询」|

### 7.3 提交流程

1. mp.weixin.qq.com → 版本管理
2. 开发版本 → 选为**体验版**（先真机测试）
3. 确认无误 → 提交**审核**
4. 填写：功能更新说明、测试账号信息、隐私政策链接
5. 等待审核（1-7 个工作日）
6. 审核通过 → 点击「**发布**」上线

### 7.4 常见审核被拒原因

| 原因 | 解决 |
|------|------|
| 功能不完整 | 确保所有页面有真实数据，无白屏/报错 |
| 未提供测试账号 | 在审核备注中注明测试账号 |
| 类目不符 | 选「教育 > 在线教育」 |
| 隐私政策缺失 | 在小程序内添加隐私政策弹窗 |
| 请求无关权限 | 检查 manifest.json，移除不必要的权限声明 |

### 生产 CORS 收严

编辑 `server/app/extensions.py`，将 CORS origins 从 `*` 改为具体域名：

```python
cors = CORS(app, resources={
    r"/api/*": {
        "origins": [
            "https://your-miniprogram.weixin.qq.com",  # 微信小程序
            "https://admin.your-domain.com"              # 管理后台
        ],
        "methods": ["GET", "POST", "PUT", "DELETE", "PATCH", "OPTIONS"],
        "allow_headers": ["Content-Type", "Authorization"],
        "supports_credentials": True
    }
})
```

## 九、运维手册

### 9.1 常用命令速查

```bash
# === 应用管理 ===
supervisorctl status eloquence        # 查看状态
supervisorctl restart eloquence       # 重启
supervisorctl stop eloquence          # 停止
supervisorctl tail eloquence          # 查看实时日志

# === 数据库备份 ===

# === 手动执行素材获取 ===

# === Nginx ===
nginx -t                              # 测试配置
systemctl reload nginx                # 重载配置

# === 查看日志 ===
tail -f /var/log/eloquence/access.log       # API 访问日志
tail -f /var/log/eloquence/error.log        # 应用错误日志
tail -f /var/log/eloquence/scheduler.log    # 素材获取日志
tail -f /var/log/nginx/eloquence-error.log  # Nginx 错误日志
```

### 9.2 监控建议

| 维度 | 方案 |
|------|------|
| 服务器监控 | 阿里云云监控（免费）|

| 小程序监控 | 微信后台「数据分析」|

### 9.3 版本更新流程

```bash
# 1. 本地代码改动 + 测试
```
