# 口才训练营 — 后台管理PC端 功能需求文档

---

## 1. 文档概述

### 1.1 基本信息

| 维度 | 内容 |
|------|------|
| 产品名称 | 口才训练营 - 后台管理系统 |
| 访问方式 | PC Web 端，浏览器访问 |
| 使用角色 | 运营管理员（一期单角色，不做RBAC） |
| 一期定位 | 满足日常运营：管素材、配打卡、调AI、看数据、推消息 |

### 1.2 技术选型（待确认）

| 决策项 | 结论 ✅ |
|--------|---------|
| 前端框架 | **Vue3 + Element Plus** |
| 构建工具 | **Vite** |
| 部署方式 | Nginx 反向代理，与 Flask 后端同服务器 |
| 登录认证 | **账号密码 + JWT** |

#### 表单字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 标题 | text | ✅ | 训练题名称，最长 50 字 |
| 分类 | select | ✅ | 4大分类下拉选择 |
| 难度 | select | ✅ | ⭐/⭐⭐/⭐⭐⭐ |
| 子分类 | select | 否 | 根据分类联动，如"演讲实战→竞聘演讲" |
| 标签 | tag-input | 否 | 回车添加，用于搜索和推荐 |
| 范文文本 | textarea | ✅ | 训练参考文本，最长 3000 字 |
| 范本音频 | file-upload | 否 | MP3/WAV，≤5MB；未上传则保存时自动调TTS生成 |
| 排序号 | number | 否 | 默认 0，数字越小越靠前展示 |
| 上架状态 | switch | — | 默认上架 |

#### TTS自动生成逻辑

```
保存训练题 → 检查是否上传了音频
  ├── 有音频 → 直接保存
  └── 无音频 → 后端异步调用阿里云TTS
        ├── 将范文文本转为MP3音频
        ├── 自动上传至OSS
        ├── 回写 training_items.sample_audio_url
        └── 列表页显示"音频生成中..." → 生成完成自动刷新
```

#### 交互规则

| 规则 | 说明 |
|------|------|
| 表单校验 | 标题/分类/难度/范文文本 4 项必填未填 → 红色边框 + 错误提示 |
| 保存后 | 关弹窗 + 列表自动刷新 + Toast"保存成功" |
| 删除 | 二次确认弹窗，删除后练习记录保留（training_item_id 置 NULL） |
| 下架 | 用户端不再展示，但已有练习记录用户仍可查看 |
| 音频替换 | 上传新音频自动替换旧音频，旧文件从OSS删除 |

## 4. 后台所需数据库表（新增）

除了小程序端已有的表，后台管理需要额外几张表：

### admin_users（管理员账号表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 主键 |
| username | VARCHAR(50) | 登录账号 |
| password_hash | VARCHAR(255) | bcrypt加密 |
| role | VARCHAR(20) | 一期固定 'super_admin' |
| last_login_at | DATETIME | 最后登录时间 |
| created_at | DATETIME | 创建时间 |

### operation_logs（操作日志表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 主键 |
| admin_id | INT FK | 操作人ID |
| action | VARCHAR(50) | 操作类型：create/update/delete/push |
| target_type | VARCHAR(50) | 操作对象：training_item/user/config |
| target_id | INT | 操作对象ID |
| detail | JSON | 操作详情（变更前后对比） |
| created_at | DATETIME | 操作时间 |

### push_records（推送记录表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 主键 |
| template_type | VARCHAR(30) | 模板类型：daily_remind/checkin_success/new_material |
| title | VARCHAR(100) | 推送标题 |
| content | TEXT | 推送内容 |
| target_count | INT | 目标用户数 |
| reach_count | INT | 实际触达数 |
| status | VARCHAR(20) | success/failed/sending |
| error_msg | VARCHAR(255) | 失败原因 |
| created_at | DATETIME | 推送时间 |

### recommend_config（推荐位配置表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 主键 |
| slot | INT | 推荐位序号 1/2/3 |
| training_item_id | INT FK | 关联训练题（可为空） |
| custom_title | VARCHAR(100) | 自定义标题（可为空） |
| created_at / updated_at | DATETIME | — |
